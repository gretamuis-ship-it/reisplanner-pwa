Connexxion Reisplanner PWA 
Een op maat gemaakte Progressive Web App voor buschauffeurs.

Functies:

1. Reisplanner
    1. Reis plannen (huidige locatie)
    2. Favoriete locaties/haltes
    3. Switch mogelijk met reisplanner voor MAT ritten?
2. Mijn diensten
    1. Koppeling met Google Spreadsheets
    2. Kijkt op een ‘slimme’ manier of een reis deze week haalbaar is (als ik om 4:30 in de ochtend moet beginnen rijden er nog geen bussen).
3. 40/52 Tracker
    1. Trackt mijn reizen, als ik een reis heb gemaakt wordt deze opgeslagen
    2. Ook functionaliteit om een reis te verwijderen (stel ik druk per ongeluk erop dat ik een reis heb gemaakt maar dat is niet zo)
    3. Indien mogelijk ook de dag en tijd registreren. Als ik later moet ‘bewijzen’ dat ik 40 weken heb gereisd kan ik dit in een oogopslag zien en een uitdraai maken in ‘Mijn OVChipkaart’
    4. Deze webapp wil ik lang kunnen gebruiken, dus elk kalenderjaar reset
4. BusMonitor
    1. Zoekt naar dichtstbijzijnde haltes
    2. Laat betreffende bussen zien die vertrekken
    3. Uiteindelijk met dienstnummer, omloop en eventueel busnummer erbij

Tech Stack: Node.js, Express, SQLite3, en een Vanilla JS frontend.
