Connexxion Reisplanner PWA 
Een op maat gemaakte Progressive Web App voor buschauffeurs.

Functies:

Lijnverkenner: Zoek haltes en geplande vertrektijden op basis van landelijke GTFS-data (SQLite).

Live Data: Integratie met OVapi voor actuele vertrektijden en vertragingen.

40/52 Tracker: Voortgangsbewaking voor het jaarlijkse doel.

Tech Stack: Node.js, Express, SQLite3, en een Vanilla JS frontend.
